# Moving Checklist

**Status:** ✅ Completed — December 2024

## Before Moving

- [x] Sign new lease
- [x] Give notice on old apartment (2 months)
- [x] Book moving company (Van der Berg Verhuizingen)
- [x] Start packing — one room at a time, 3 weeks early
- [x] Label every box: room + contents
- [x] Redirect mail via PostNL

## Moving Day

- [x] Final walkthrough of old apartment
- [x] Take photos of old place before handing over keys
- [x] Hand over keys to landlord
- [x] Movers: 9am start, done by 2pm

## After Moving

- [x] Update address: bank, insurance, employer, GP, dentist
- [x] Set up internet (took 5 business days)
- [x] Register new address at municipality
- [x] Unpack essentials box first (sheets, towels, coffee, charger, toilet paper)
