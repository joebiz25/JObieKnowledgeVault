Navigation
- `Space` or `f` = next page
    
- `b` = previous page
    
- `/pattern` = search for text
    
- `n` = next search result
    
- `q` = quit

Don't know a command?
Use man -k search phrase (where search phrase is some word that relates to the command e.g. copy)

help is the man command for shell built in commands cd

When to use which information resource:

-->man find          # The find command has many options

-->help read         # read is a bash builtin

--> grep --help       # Quick overview of grep options

-->man -k permission # Find commands related to permissions
