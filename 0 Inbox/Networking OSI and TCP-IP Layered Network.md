
OSI is 7 layered (use mnemonic: All People Some Time Need Data Processing): Open  Systems Interconnect composed of the following layers:
Application
Presentation
Session
Transport
Network
Datalink
Physical

TCP/IP is 4 layered (groups some of the layers of OSI): Also called Internet Network Model
Application (OSI: Application, Presentation, Session)
Transport
Datalink (OSI: Network and Datalink)
Physical

OSI has been discarded as the norm for describing or modeling networks. 
Internet Network Model is the widely recognized and accepted model.

However, OSI is still used widely in the IT world especially by vendors and manufacturers for specifying at which layer Networking products operate
